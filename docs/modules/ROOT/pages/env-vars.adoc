= Configure Environment Variables

You can configure environment variables in the `hazelcast` pods using the `env` field of the CRD. The `env` field uses the link:https://kubernetes.io/docs/reference/generated/kubernetes-api/v1.27/#envvar-v1-core/[Kubernetes' EnvVar format].

WARNING: You cannot use the `env` field to configure Hazelcast environment variables, which are prefixed `HZ_`. To configure Hazelcast environment variables, use xref:custom-config.adoc[].
