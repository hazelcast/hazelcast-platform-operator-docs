= Configure Management Center JVM arguments
:description: Custom JVM arguments in Hazelcast Management Center allow for the precise configuration of Java runtime parameters, enabling you to optimize resource utilization and enhance the overall performance of your Hazelcast cluster management. It can be useful to pass JVM arguments, such as for configuring log levels or supporting the IPv6 stack.

{description}

The structure of configuring JVM arguments in the Management Center CR is same as Hazelcast CR. See xref:jvm-parameters.adoc[].
